Setup Instructions
  1. Get a Google OAuth Client ID from Google Cloud Console
  2. Replace 'YOUR_GOOGLE_CLIENT_ID_HERE' in assets/js/auth.js
  3. Set up Google Sheets with 'Students' and 'Transactions' sheets
  4. Deploy the Google Apps Script from google-apps-script-api.js
  5. Update API URLs in all JS files
  6. Add approved email addresses to the allowedEmails array in auth.js
